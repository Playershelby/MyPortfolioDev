    Snake Game
    🐍

    Descubra Experiências Inesquecíveis

    O Snake Game é um projeto desenvolvido em Python utilizando a biblioteca Pygame, com o objetivo de recriar de forma divertida e moderna o clássico jogo da cobrinha.
    A lógica do jogo é simples: o jogador controla a cobra, que deve comer os alimentos que aparecem na tela para crescer, evitando colidir com as paredes ou com o próprio corpo.
    O código foi estruturado com uma lógica leve e eficiente, ideal para rodar em qualquer ambiente local. Além disso, o jogo é divertido e viciante, trazendo aquela sensação nostálgica dos games retrô.

    O projeto foi desenvolvido com HTML, CSS e JavaScript, explorando as principais tecnologias do front-end para criar uma experiência leve e interativa.
